# neural-net-backpropagation-matlab
 
Data file should be in the form

|       |       |     |       |     |
|-------|-------|-----|-------|-----|
| x_1,1 | x_1,2 | ... | x_1,D | y_1 |
| x_2,1 | x_2,2 | ... | x_2,D | y_2 |
| ...   | ...   | ... | ...   | ... |
| x_N,1 | x_N,2 | ... | x_N,D | y_N |

Change L, Nh, Nit, num_val_sets if needed. Should also work in GNU Octave, just slower.
